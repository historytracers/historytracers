# CSV

Este diretório contém arquivos GEDCOM vinculados para os leitores na seção de `Árbol Genealógico`.

**Nota**: Por favor, não adicionem arquivos a este diretório manualmente. Atualmente, esses arquivos são gerados utilizando o script `scripts/python/csv_to_gedcom.py` que usa o software [Gramps](http://gramps.org/) através da importação de arquivos CSV.
