## CSV

Este directorio contiene archivos GEDCOM que se enlazan para los lectores en la sección del `Árbol Genealógico`.

**Nota**: Por favor, no añadan archivos a este directorio manualmente. Actualmente, estos archivos se generan utilizando el software [Gramps](http://gramps.org/) mediante la importación de archivos CSV. La integración completa del sistema aún está en desarrollo.
