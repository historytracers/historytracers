## CSV

Este directorio contiene archivos GEDCOM que se enlazan para los lectores en la sección del `Árbol Genealógico`.

**Nota**: Por favor, no añadan archivos a este directorio manualmente. Actualmente, estos archivos se generan utilizando el script `scripts/python/csv_to_gedcom.py` que usa [Gramps](http://gramps.org/).
